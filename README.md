# Hydrostatic Force Simulator & Calculator 🌊

A high-precision technical tool developed in **Python** to calculate and simulate hydrostatic forces on submerged surfaces. This project combines complex physics engineering with a modern, user-friendly interface.

## 🚀 Core Functionalities
* **Real-time Calculations:** Instant results for hydrostatic force, pressure center, and fluid density variables.
* **Engineering Logic:** Implements fundamental principles of fluid mechanics with high mathematical accuracy.
* **Modern UI:** Built with **CustomTkinter** for a professional, dark-mode sleek desktop experience.
* **Data Validation:** Integrated checks to ensure physical parameters are within realistic bounds.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **GUI Library:** CustomTkinter (Modernized Tkinter wrapper)
* **Math Engine:** Python Standard Math Library
